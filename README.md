# COSMIC

GEANT4-моделирования эксперимента Дейтрон.
Внесена геометрия установки из 2021 и 2023 года. Изначально писалась для космической калибровки детектора.
Позже были добавлены другие генераторы процессов.

1. Космика
2. pn-пара из кинематики
3. процесс для LQ-поляриметра для дейтериевой и водородной мишени. Квазиупругое рассеяние ep в модели спектатора
   Процесс ed(сигнал)+ep(фон)
4. Процессы из генератора GENBOSS, в частности сигнальный $ \gamma D = pn $, для анализа фона.
5. pp-пара для калибровки протонной дорожки.
6. Процесс $ \gamma D = pp\pi^- $ также из GENBOSS

Реализованы различные варианты триггера для записи в файл как в эксперименте.
Поддержка многопоточности и вывод в единый root файл информации с детекторов по завершении симуляции.
